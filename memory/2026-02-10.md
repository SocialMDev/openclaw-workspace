## Token Monitor Skill Created

Built a custom token monitoring skill for OpenClaw to analyze cross-session token usage.

**Location:** `skills/token-monitor/`
- `SKILL.md` - Documentation
- `token-monitor.js` - Main script

**What it does:**
- Parses `~/.openclaw/agents/*/sessions/sessions.json`
- Shows per-session token counts (input/output/total/context)
- Calculates cost estimates
- Ranks top token-burning sessions
- Shows distribution stats (avg, median, percentiles)

**Current usage findings:**
- 57 sessions tracked
- 1.26M total tokens (1.24M input + 20K output)
- Estimated cost: $3.30
- Heaviest session: `agent:main:main` at 226K tokens (18% of total)
- Many cron job sessions accumulating 10-30K tokens each

**Usage:**
```bash
node skills/token-monitor/token-monitor.js [--limit N] [--detail] [--days N]
```
